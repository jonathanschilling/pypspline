PyPspline

A Python interface to the Fortran spline interpolation library PSPLINE.

PyPpspline is Python module for interpolating and computing derivatives of fields in up to three dimensions. Boundary conditions, which are by defaults of type not-a-knot, can be periodic, or prescribe the normal first or second derivative at the boundary. Function value and derivatives can be computed on isolated points (point interpolation), on a list of scattered points (cloud interpolation) or on a structured grid (array interpolation).

  • Prerequisites:
    • a C compiler (for instance gcc)
    • a Fortran 90 compiler (to build PSPLINE).
    • NetCDF
    • Python
    • NumPy
    • ScientificPython (to read and write NetCDF files)
